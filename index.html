食事半功倍 — 后端服务

提示： 此服务仅面向微信小程序端调用，请使用 wx.cloud.callContainer 访问 API。

可用 API

GET /api/ping — 健康检查
GET /api/count — 获取计数器
POST /api/count — 更新计数器
GET /api/wx_openid — 获取微信 OpenID
POST /api/coze/ask — Coze AI 对话
POST /api/diet — 保存饮食记录
GET /api/diet?date=YYYYMMDD — 获取饮食记录
POST /api/training — 保存训练日志
GET /api/training?date=YYYYMMDD — 获取训练日志
POST /api/points — 增减积分
GET /api/points — 获取积分
POST /api/weight — 保存体重
GET /api/weight?days=N — 获取体重记录
